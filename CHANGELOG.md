## [0.1.0] - 2026-01-21

### Added

- Initial protocol specification
- Core concepts documentation (`concepts.md`)
- Capability model (`capabilities.md`)
- Data and state access (`data-state.md`)
- Action model (`actions.md`)
- View layer specification (`views.md`)
- Permission model (`permissions.md`)
- Versioning guidelines (`versioning.md`)
- JSON Schema for provider manifests (`manifest.schema.json`)
- Example provider manifests:
  - Media Catalogue (`examples/media-catalogue.json`)
  - Smart Lights (`examples/smart-lights.json`)
  - Video Feed (`examples/video-feed.json`)

### Notes

- This is the initial draft release
- The protocol is subject to change based on implementation feedback
- Feedback and contributions welcome via GitHub Issues
